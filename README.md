# Crypto Tax Calculator

Crypto Tax Calculator is a tool designed to help cryptocurrency traders and investors calculate their tax liabilities based on their trading activities. 

## Project Structure
```
taxtim/
├── backend/              # Laravel API
│   ├── app/
│   ├── routes/
│   └── ...
└── frontend/             # React App
    ├── src/
    ├── public/
    └── ...
```
